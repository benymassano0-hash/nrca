# ⚡ INICIAR RCNA - Quick Start

## 🎯 Forma Mais Rápida

### Clique duplo neste arquivo:

```

```

Aguarde 10 segundos e o navegador abre automaticamente em:
```
http://localhost:3000/login
```

1. Clique em **"Registre-se aqui"**
2. Preencha o formulário
3. Clique **"Registar"**

## 🆘 Se `iniciar.bat` não funcionar

Use Terminais Manuais:

**PowerShell 1:**
```powershell
cd "c:\Users\josé massano\bb\RCNA\backend"

```

```powershell

```
